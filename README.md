# SoulFix

Интуитивный и кастомизируемый плагин для починки предметов через графический интерфейс (GUI) с поддержкой динамических анимаций, экономики и гибкой системы слотов.

## Описание
**SoulFix** предоставляет игрокам удобный и сбалансированный способ ремонта снаряжения. Вместо моментальной починки процесс оформлен в виде красивой анимации. Плагин позволяет ограничивать базовое количество доступных слотов для ремонта, а также даёт возможность докупать новые слоты за внутриигровую валюту с прогрессирующей ценой.

---

## Особенности
* **Интерактивный GUI** — интуитивно понятное меню для размещения поврежденных предметов.
* **Экономика и покупка слотов** — поддержка Vault и PlayerPoints для покупки дополнительных слотов ремонта с гибким ростом цены.
* **Эффекты и анимация** — процесс починки сопровождается кастомными звуками, частицами (particles) и отображением прогресса в BossBar.
* **Умные кулдауны** — время перезарядки гибко масштабируется в зависимости от количества отремонтированных предметов.
* **Система лимитов по группам** — настройка стартового количества слотов и лимита покупных ячеек через права (permissions).
* **Интеграция с PlaceholderAPI** — плейсхолдеры для вывода информации о слотах и кулдаунах на экране игрока.

## Команды
* `/soulfix` — открыть графическое меню починки предметов.
* `/soulfix slots` — посмотреть информацию о своих слотах (базовые, купленные, доступный лимит).
* `/soulfix admin` — показать список команд администратора.
* `/soulfix admin reload` — перезагрузить конфигурацию и языковые файлы без перезапуска сервера.
* `/soulfix admin giveslots <игрок> <количество>` — выдать игроку дополнительные купленные слоты.
* `/soulfix admin removeslots <игрок> <количество>` — забрать у игрока купленные слоты.
* `/soulfix admin resetcooldown <игрок>` — мгновенно сбросить время перезарядки для указанного игрока.

---

## Права
* `soulfix.use` — базовый доступ к функционалу плагина и меню починки.
* `soulfix.admin` — доступ ко всем административным функциям и командам.
* `soulfix.bypass.cooldown` — полный обход задержки (кулдауна) на починку.
* `soulfix.slots.default` / `vip` / `premium` / `legend` — права, определяющие стартовое количество доступных слотов для ремонта и максимальный лимит на их покупку.
